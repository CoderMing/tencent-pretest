## 项目说明

脚手架：`@vue/cli`

CSS预处理器：`stylus`



​	自己 Mook 了接口，可以在 `src/mooks/` 文件夹中查看 Mook。

​	尝试使用了一些Vue的功能，例如 `Filter`，组件分离等。

​	`stylus` 使用了一部分自定义函数，可以在 `src/libs/util.styl` 文件中查看。

​	下半页设置高度大于100vh，目的是为了能让页面滑动，让顶部菜单栏能看到 fixed 效果
